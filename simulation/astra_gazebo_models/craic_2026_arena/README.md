# CRAIC 2026 Arena Assets

这套模型资源用于 CRAIC 2026 比赛场地：

- `craic_2026_arena`
  基础场地与固定道具，不含可变位置圆环
- `craic_2026_ring`
  单独的圆环模型，由 world 在启动时决定具体 `x` 位置

仿真坐标系做了一个任务执行友好的平移。生成后的 world 满足：

```text
x_world = x_rule - 1.5
y_world = y_rule - 3.0
```

也就是：

- 比赛规则里的起飞点 `(1.5, 3.0)` 会被映射成 Gazebo / PX4 的出生点 `(0, 0)`
- 飞机默认 `yaw=0` 时，机头正前方就是“二维码 -> 障碍物 -> 任务区”的主飞行方向
